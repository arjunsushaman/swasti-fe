## Image Guidelines

- **Format:** JPG or PNG (JPG recommended for smaller file size)
- **Dimensions:** At least 400x400px (square ratio preferred)
- **Size:** Keep under 200KB for optimal loading
- **Quality:** High-resolution professional headshots
- **Background:** Clean, professional background preferred

## Notes

- Images will be displayed as circular avatars
- Make sure the subject's face is centered in the photo
- If an image is missing, a placeholder icon will be shown instead
- You can also use PNG format by changing the extension in `/lib/constants.ts`
